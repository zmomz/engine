# Keyboard Shortcuts Reference

## Navigation

| Shortcut | Action |
|----------|--------|
| `Alt + 1` | Navigate to Overview |
| `Alt + 2` | Navigate to Dashboard |
| `Alt + 3` | Navigate to Positions |
| `Alt + 4` | Navigate to Queue |
| `Alt + 5` | Navigate to Risk |
| `Alt + 6` | Navigate to Analytics |
| `Alt + 7` | Navigate to Settings |

## Global Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + R` | Refresh current page data |
| `Escape` | Close mobile sidebar |
| `Escape` | Dismiss alerts/modals |
| `Tab` | Navigate through interactive elements |
| `Shift + Tab` | Navigate backwards through elements |

## Data Tables

| Shortcut | Action |
|----------|--------|
| `Arrow Keys` | Navigate table cells |
| `Page Up/Down` | Scroll table pages |
| `Home` | Go to first row |
| `End` | Go to last row |
| `Enter` | Activate selected row action |

## Risk Management

| Shortcut | Action |
|----------|--------|
| `F` | Force Start queue (when on Risk/Overview page) |
| `S` | Force Stop queue (when on Risk/Overview page) |
| `E` | Run manual risk evaluation |

## Queue Management

| Shortcut | Action |
|----------|--------|
| `P` | Promote selected signal |
| `Delete` | Remove selected signal |
| `Ctrl/Cmd + A` | Select all signals |

## Accessibility

| Shortcut | Action |
|----------|--------|
| `Alt + /` | Show keyboard shortcuts help |
| `Ctrl/Cmd + =` | Zoom in |
| `Ctrl/Cmd + -` | Zoom out |
| `Ctrl/Cmd + 0` | Reset zoom |

## Browser

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Shift + Delete` | Clear browser cache |
| `F5` | Reload page |
| `Ctrl/Cmd + Shift + R` | Hard reload (bypass cache) |
| `F11` | Toggle fullscreen |
| `F12` | Open developer tools |

## Tips

1. **Mobile Navigation**: On mobile devices, use the hamburger menu icon in the top-left to access the sidebar.

2. **Quick Refresh**: Press `Ctrl/Cmd + R` on any page to force refresh the data without reloading the entire page.

3. **Focus Mode**: Press `F11` to enter fullscreen mode for a distraction-free trading experience.

4. **Data Tables**: Click column headers to sort. Hold `Shift` while clicking for multi-column sorting.

5. **Filters**: Use the filter icon in table toolbars to quickly filter data without scrolling.

6. **Confirmation Dialogs**: All destructive actions (Force Close, Remove Signal) require confirmation. Press `Enter` to confirm or `Escape` to cancel.

## Notes

- Shortcuts may vary slightly between browsers
- Some shortcuts require focus on the specific page/component
- Custom shortcuts can be configured in Settings (future feature)
- Screen readers: All actions have proper ARIA labels for accessibility
